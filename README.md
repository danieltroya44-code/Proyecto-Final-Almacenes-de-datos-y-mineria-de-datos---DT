# Predicción de Deserción Estudiantil

Este proyecto implementa un sistema de predicción de deserción estudiantil utilizando técnicas de minería de datos y Machine Learning.

## Tecnologías
- Python
- Scikit-learn
- Pandas
- Streamlit

## Estructura
- app.py → Aplicación Streamlit
- entrenamiento.ipynb → Entrenamiento del modelo
- modelo_desercion.pkl → Modelo entrenado
- scaler.pkl → Escalador
- informe.pdf → Informe técnico

## Ejecución
Instalar dependencias:
pip install streamlit
pip install numpy
pip install pandas
pip install joblib
pip install matplotlib
pip install seaborn
pip install scikit-learn
pip install openpyxl

# Ejecutar la aplicación:
streamlit run app.py
